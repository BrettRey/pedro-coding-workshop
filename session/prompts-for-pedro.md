# Prompts for Pedro

Plain-language prompts for the live demo. Brett types these during the session; Pedro watches and comments via Zoom.

## Opening prompts (warm-up)

1. **Format discovery:**
   "Read these Chuj transcription files and describe the format."

2. **Quick inventory:**
   "Which children are in this dataset? What are their ages? How much data is there for each one?"

## Core task: morphological extraction

3. **Ergative markers:**
   "Extract all Set A (ergative) person markers from the child utterances and count them by child. Set A markers in Chuj are prefixes: hin- (1SG), ha- (2SG), s-/y- (3SG), ko- (1PL). Search in the corrected adult form (the = line) for child utterances only."

4. **Correction (after Pedro flags false positives):**
   "Remove Spanish words like yo, ya, hay, and han from the counts. Exclude onomatopoeia too."

## Follow-up prompts (if time)

5. **Longitudinal:**
   "Compare Mateo's Set A marker use across his four recording sessions. Does the frequency change over the month?"

6. **Questions:**
   "Find all questions in the data. How many questions does each child ask? What question words do they use?"

7. **Code-switching:**
   "Find all utterances where the child uses a Spanish word inside a Chuj sentence. Which Spanish words are most common?"

8. **Word counts:**
   "Count the average utterance length (in words) for each child. Do older children produce longer utterances?"

## Prompts Pedro might want to try on his own data later

- "Read my Q'anjob'al transcription files and find all instances of aspect markers."
- "Count how often each child uses the completive vs. incompletive aspect."
- "Find all utterances containing a transitive verb and check whether the ergative marker agrees with the subject."
- "Make a table of the most frequent verbs each child uses, with Spanish translations."
